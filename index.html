Sobre o Repertório Fácil

Versão 1.0 • Documento oficial do produto

Menos cliques. Mais música.

O Repertório Fácil foi criado para ajudar bandas, músicos, ministérios e projetos musicais a organizar integrantes, músicas, repertórios e eventos em um só lugar.

Missão

Criar o aplicativo mais simples e intuitivo para gestão de bandas. O foco não é ter mais botões, e sim economizar tempo para quem ensaia, toca e organiza apresentações.

Fluxo principal

  • Login
  • Meus Projetos
  • Dashboard do projeto
  • Integrantes, Músicas, Repertórios e Eventos

Funcionalidades atuais

  • Cadastro de projetos musicais
  • Cadastro de integrantes
  • Biblioteca de músicas
  • Montagem de repertórios
  • Eventos vinculados a repertórios
  • PDF e compartilhamento

Princípios de desenvolvimento

  • Não complicar o uso para o músico.
  • Consolidar cada módulo antes de avançar.
  • Manter visual consistente em todas as telas.
  • Entregar arquivos completos a cada atualização.

Próximas fases

  • Polimento final da interface
  • Modo Show
  • QR Code
  • Agenda inteligente
  • Dashboard com dados do projeto
